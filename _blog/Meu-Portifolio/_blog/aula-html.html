Soft Skills

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na web. documentos HTML podem ser interpetrados por navegadores.
Nível: Intermediário

CSS

o CSS permite controlar a aparência e o layout de um site, facilitando a criação de páginas da web visualmente atraentes e responsivas. Ele funciona aplicando regras de estilo a elementos HTML.
Nível: Intermediário

JAVASCRIPT

JavaScript é usado para criar e manipular elementos de página web, controlar comportamentos interativos, validar formulários, criar animações e muito mais.
Nível: Básico

GIT

o Git é uma ferramenta essencial para o desenvolvimento de software colaborativo, oferecendo controle preciso sobre o código-fonte e facilitando a colaboração entre desenvolvedores.
Nível: Essencial

GITHUB

o GitHub é uma plataforma essencial para o desenvolvimento de software colaborativo, fornecendo uma infraestrutura robusta para hospedar, colaborar e compartilhar código-fonte.
Nível: Essencial

C#

C# é uma linguagem de programação moderna e poderosa, amplamente utilizada para desenvolver uma ampla gama de aplicativos em diferentes plataformas, com uma forte ênfase em segurança, desempenho e produtividade do desenvolvedor.
Nível: Básico

My SQL

O MySQL é um sistema de gerenciamento de banco de dados usado para armazenar e organizar dados de forma eficiente. Ele usa a linguagem SQL para consultar e manipular esses dados.
Nível: Básico
